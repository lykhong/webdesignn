Ví dụ các thẻ HTML

1. HTML Basic

Thẻ cơ bản để khai báo một tài liệu HTML bao gồm: <!DOCTYPE html>, <html>, <head>, <body>.

2. HTML Elements

Một phần tử HTML bao gồm một thẻ mở, nội dung và thẻ đóng.

<p>Đây là một đoạn văn bản</p>

3. HTML Attributes

Thuộc tính HTML cung cấp thêm thông tin về phần tử, ví dụ như href trong thẻ <a>.

<a href="https://www.google.com">Google</a>

4. HTML Headings

Các thẻ <h1> đến <h6> dùng để tạo tiêu đề.

<h1>Tiêu đề lớn</h1>
<h3>Tiêu đề nhỏ hơn</h3>

5. HTML Paragraphs

Thẻ <p> dùng để tạo đoạn văn bản.

<p>Đây là một đoạn văn bản</p>

6. HTML Styles

Dùng thuộc tính style để thêm kiểu cho phần tử.

<p style="color:red;">Chữ màu đỏ</p>

7. HTML Formatting

Các thẻ như <b>, <i>, <u> dùng để định dạng văn bản.

<b>Đậm</b>, <i>Nghiêng</i>, <u>Gạch chân</u>

8. HTML Quotations

Thẻ <blockquote> dùng để trích dẫn một đoạn văn bản.

<blockquote>Đây là một trích dẫn</blockquote>

9. HTML Comments

Chú thích HTML giúp ghi chú trong mã mà không hiển thị trên trình duyệt.

<!-- Đây là chú thích -->

10. HTML Colors

Dùng thuộc tính style để thay đổi màu sắc.

<p style="color:blue;">Chữ màu xanh</p>

11. HTML Links

Thẻ <a> tạo liên kết đến trang khác.

<a href="https://www.example.com">Click vào đây</a>

12. HTML Images

Thẻ <img> dùng để hiển thị hình ảnh.

<img src="image.jpg" alt="Hình ảnh">

13. HTML Tables

Thẻ <table> tạo bảng.

<table border="1">
  <tr>
    <th>Tiêu đề</th>
    <th>Cột 2</th>
  </tr>
  <tr>
    <td>Dữ liệu 1</td>
    <td>Dữ liệu 2</td>
  </tr>
</table>

14. HTML Lists

Thẻ <ul> và <ol> tạo danh sách không thứ tự và có thứ tự.

<ul>
  <li>Mục 1</li>
  <li>Mục 2</li>
</ul>

15. HTML Block & Inline

Các thẻ block như <div> chiếm toàn bộ chiều rộng, trong khi inline như <span> chỉ chiếm nội dung của nó.

<div style="background:lightgrey;">Thẻ block</div>
<span style="background:yellow;">Thẻ inline</span>
